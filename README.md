# AR Tours WhatsApp Customer Care Agent

Professional WhatsApp customer care agent powered by Claude AI for tour operations.

## Quick Start

1. Copy `.env.example` to `.env`
2. Fill in your WhatsApp credentials
3. Run `npm install`
4. Deploy to Render

## Features

- 24/7 automated tour booking support
- AI-powered responses
- Webhook-based message processing
- Production-ready Node.js/Express backend
